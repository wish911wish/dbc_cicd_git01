# メモ

```

|分類|処理内容の概要(約50字)|テスト観点(約100字)|

|参照処理|SELECT文を実行し、抽出条件・結合・集計を経てソースからデータを取得する処理|抽出条件・結合・集計が仕様通りで、期待した件数と値が正しく取得できることを検証する。並び順、結果の列構成・型、該当データ0件時の挙動、結合による意図しない行数変動やNULLの扱いも確認する。
|整形処理|取得データに対し型変換・トリム・空値のNULL化などのクレンジングを行う処理|型変換・トリム・空値のNULL化が仕様通りに適用され、変換後の値・型が期待通りであることを検証する。NULLや空文字、境界値、変換不能な異常値などの入力に対し、意図した結果や例外挙動になることも確認する。|
|その他|参照・整形に属さない共通処理や補助処理(接続・ログ・パラメータ制御など)|各処理が想定通りに動作し、異常入力やエラー時に適切な例外・戻り値・ログ出力となることを検証する。処理の呼び出し前提や依存関係が満たされない場合の挙動も確認する。|

```
